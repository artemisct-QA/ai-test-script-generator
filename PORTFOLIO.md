# 🎓 Portfolio: AI Test Script Generator

## Project Overview

During **Global Hack Week 2026 - GenAI Track**, I built an AI-powered test script generator that auto-generates production-ready test automation code from API specifications using Claude AI.

## 🎯 The Problem

As an SDET, I identified a real pain point:
- Manual test writing is repetitive and time-consuming
- Teams duplicate boilerplate code across projects
- Engineers spend hours on setup instead of strategy

## ✨ The Solution

A tool that:
- Takes an API specification as input
- Generates 200+ lines of production-quality test code in seconds
- Supports **both Python (pytest) and JavaScript (Jest)**
- Includes proper mocking, fixtures, and comprehensive scenarios
- Reduces test boilerplate by ~70%
